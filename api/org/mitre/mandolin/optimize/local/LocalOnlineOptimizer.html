LocalOnlineOptimizer

class LocalOnlineOptimizer[T, W <: Weights[W], LG <: LossGradient[LG], U <: Updater[W, LG, U]] extends LocalOptimizerEstimator[T, W]

Provides the functionality of the distributed OnlineOptimizer without using Spark/RDDs to provide non-Spark training that supports the same classifier API.
