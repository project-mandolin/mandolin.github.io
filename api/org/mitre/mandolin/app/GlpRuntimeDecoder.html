GlpRuntimeDecoder

class GlpRuntimeDecoder extends AnyRef

A simple runtime decoder for trained GLPs. Provides a lightweight way to use a trained classifier from Scala or Java by providing sparse feature vectors as a simple string or list of strings. Short of a full Java API, this provides a simple method that takes in a Java or Scala list of strings as a sparse feature representation and returns a single floating point corresponding to the posterior probability associated with the positive class.
